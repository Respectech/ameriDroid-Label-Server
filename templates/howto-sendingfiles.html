How to Send Files to the Printer

Using this web interface is not the only way to print automatically. Files on your computer can be copied to this server and will be printed automatically. The instructions on how to do so -- and helpful information -- is included below.

Windows

Drag-and-drop

  • Open File Explorer
  • In the address bar type \\1.2.3.4\print and press ENTER (replace 1.2.3.4 with the IP address of the printer, shown at the top of this browser page)
  • The default username and password are both "label" (lowercase, without the quotes)
  • Files can be copied to this location
  • IMPORTANT: Files will be deleted from this directory after printing, so make sure to have a copy of them

Saving directly

  • From any application, select "Save as", "Save as File", "Save as PDF", "Export as PDF" or similar.
  • In the address bar, type \\1.2.3.4\print and press ENTER (replace 1.2.3.4 with the IP address of the printer, shown at the top of this browser page)
  • If prompted for a password, the default username and password are both "label" (lowercase, without the quotes)
  • The filename doesn't matter - only the extension (.pdf, .jpg, etc.)

Accepted File Types

  • PDF files, including multi-page. Multi-page PDFs will have pages stacked vertically, one after the other, as one long image.
  • Many image file types.
  • If an unsupported file type is saved to this directory, it will be automatically deleted with no action being taken.
  • NOTE: If the image is wider than tall, it will be rotated and printed with the shorter side scaled to the width of the printer paper, and the height adjusted to retain the aspect ratio.
  • IMPORTANT: If the image is very short, the width will be expanded to retain the aspect ratio by the same proportion and could result in a very long print. Make sure to force the dimensions of the image to a reasonable number using the "Special Features" listed below.

Special features

The following can be added to the filename to cause specific behavior when printing the file:

  • +ws - Does not trim whitespace from around the image/page. If this is not present, whitespace will be trimmed.
  • -gs - Will print in solid colors (black) instead of grayscale. If this is not present, grayscale will be used.
  • |w=123| - Will force the image to be printed 123 pixels wide. The height (if not defined) will automatically be adjusted to retain the aspect ratio.
  • |h=456| - Will force the image to be printed 456 pixels tall. The width (if not defined) will automatically be adjusted to retain the aspect ratio.
  • NOTE: If the width and height are both specified, the image will be stretched to fit both of those values, and will not retain the aspect ratio.
  • IMPORTANT: The default behavior of this software is to orient files so they will print in the largest format.
