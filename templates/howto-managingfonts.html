How to Manage Fonts

Windows

Drag-and-drop

  • Open File Explorer
  • In the address bar type \\1.2.3.4\fonts and press ENTER (replace 1.2.3.4 with the IP address of the printer, shown at the top of this browser page)
  • The default username and password are both "label" (lowercase, without the quotes).
  • .ttf font files (Truetype Fonts) can be copied to this location.
  • If there are font files you don't want, you can delete them from this location.
  • Click the "Restart Webserver" in the menu to re-scan the font folder after adding the font.
  • NOTE: Fonts are displayed in alphabetical order.
  • NOTE: Fonts are validated to make sure they do not contain fatal errors before being included in the list of available fonts.
  • NOTE: A source we've used successfully for free online fonts is https://www.dafont.com.
